Ozone concentration

Given the air quality dataset, Ill try to predict the amount of Ozone (in parts per billion) using a random forest. The user can generate predictions with this dataset using the interactive shiny application. Since the model is trained on application start, the initial loading may take some time. The shiny app can be found here.

Exploratory plot

Looking at the pairs plot, Ozone seams negatively correlated with the wind, and positively with the temperature. Solar Radiation also seems somewhat positively correlated.

Training the model

The model is built using all variables in the dataset. Since it is rather small, lets use the bootstrapping method in train control. Looking at the feature importance, it seems our original exploratory data analysis matches.

trc <- trainControl(method="boot")
rf <- train(Ozone~., data=na.omit(airquality), method='rf', 
            metric="RMSE", trControl=trc)
importance(rf$finalModel)

Example prediction

Lets try predicting something for the median values of all parameters.

df <- data.frame(median(na.omit(airquality$Solar.R)), 
                 median(airquality$Wind), median(airquality$Temp), 
                 median(airquality$Month), median(airquality$Day))
colnames(df) <- c('Solar.R', 'Wind', 'Temp', 'Month', 'Day')
predict(rf, df)
